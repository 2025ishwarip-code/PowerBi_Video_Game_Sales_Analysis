#PowerBi_Video_Game_Sales_Analysis

#Overview
Power BI dashboard analyzing video game sales with region-wise and global insights using interactive visuals and KPI-driven analysis.

#Report pages
This Power BI file contains the following pages:
-Sales Data
-NA_Sales
-EU_Sales
-JP_Sales
-Other Sales
-Global Sales

#Key Features
-KPI Cards for quick performance summary
-Slicers for interactive filtering 
-Line charts to show sales trend over time
-Clustered Column chart to show sales by genre
-Pie chart for showing sales by top 5 publishers
